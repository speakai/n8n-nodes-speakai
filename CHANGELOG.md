# Changelog

## 0.1.0

Initial release.

- **Speak AI** action node with four operations: upload a file to transcribe and analyze, upload a text note to analyze, run a magic prompt, and find media by ID.
- **Speak AI Trigger** node that registers a webhook with Speak AI and fires on new automated transcriptions, captions, sentiment analysis, magic prompt responses, and recordings.
- **Speak AI API** credential (server URL + API key).
